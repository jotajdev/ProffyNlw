👷**Comandos**

✔️* Instalar o Expo-cli:

`npm install -g expo-cli`

✔️* Inicializar uma aplicação

`expo init name_project`

✔️* Instalar dependências

`npm install`

✔️* Start na aplicação

`npm start`
